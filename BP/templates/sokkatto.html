Where did name Sokkatto come from?

The exact origin of this name is unknown however it does come from
one of my older names which were not mentioned in an introduction.
Sokkatto became sort of side character for me.

Why is it so important to me?

The name Sokkatto was given to my current biggest project. It is a
rust+ / discord bot which converts information from app rust+ into
graphical form.

So how does it work?

The game Rust recently added the option to pair server with mobile
app rust+. This app provides information such as in-game time, season,
team chat messages but also map, drone view and much more.

The reason it is such a deal is that I managed to convert all of this
info and use it in discord via discord.py API. Then, all of this is
to webapp which displays all information from rust+ but also has
options like CPU and Memory usage.

Preview of camera

This is one of all the things that can be done. A camera is watching a
scene, rendering every update everything in frame (players and trees at
the moment).
